Overview:
A Power BI analytics project that explores the World Happiness Report dataset to quantify how factors like economic output, social support, and life expectancy influence subjective well-being. The dashboard transforms raw global data into interactive visual stories, highlighting the disparity between the happiest and least happy nations.

Key Features & Insights:

Global Overview: Tracks key metrics across 139 countries, identifying Finland (7.74) as a top leader in global happiness.

Regional Benchmarking: Categorizes performance by continent, showing how regions like North America and Europe compare against Africa and Asia in happiness scores.

Factor Analysis: Breaks down happiness into its core drivers, including:

GDP per capita

Social Support

Healthy Life Expectancy

Freedom to make life choices

Perceptions of Corruption

Detailed Country Profiles: Drill-down capabilities to view specific stats, such as the United States' score breakdown (6.94) and its primary drivers.

Technical Tools Used:

Power BI: For data modeling, DAX calculations, and dashboard design.

Custom Visuals: Integrated heatmaps and specialized charts (like Icon Map Pro) to visualize regional impacts and factor correlations.

Data Source: World Happiness Report (annual global surveys)
